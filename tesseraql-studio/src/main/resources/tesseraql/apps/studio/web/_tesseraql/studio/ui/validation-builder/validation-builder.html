Field domains

Single-field constraints (type, bounds, pattern, enum) belong in a field domain referenced from input: — this builder is for cross-field and SQL rules that a domain cannot express.

Validation builder

Generate a route's validate: block for one rule, then copy it into a command route. Expression rules use the core expression language (&& || == != < > <= >=); a SQL rule points at a validation SQL file whose returned rows are the violations.
