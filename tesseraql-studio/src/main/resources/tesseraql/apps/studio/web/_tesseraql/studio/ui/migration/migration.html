New migration

Creates a Flyway migration under db/…/migration. A versioned migration is auto-numbered (next: V1 for the main datasource); a repeatable one is R__<name>. The new file needs a restart + migrate to be applied — the running app only lists it. Flyway has no built-in undo on the free edition: roll back by writing a follow-up migration (fix-forward).
